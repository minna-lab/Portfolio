# Portfolio — ASSOSSOU Minna

Un portfolio personnel modern et élégant créé avec HTML, CSS et JavaScript. Ce projet met en avant les compétences, projets et expériences professionnelles d'une étudiante en Bachelor Informatique.

## 🎯 Caractéristiques

- ✨ Design moderne et responsive
- 🌙 Mode clair/sombre (thème de couleur)
- 📱 Entièrement adapté mobile et desktop
- ⚡ Navigation fluide avec scroll smooth
- 🎨 Animations élégantes et transitions CSS
- 💌 Section contact avec formulaire email
- 📊 Affichage des compétences avec barres de progression
- 🏆 Timeline des diplômes et expériences
- 🔗 Liens vers les projets GitHub

## 🛠️ Technologies Utilisées

- **HTML5** - Structure et contenu
- **CSS3** - Styling et animations
- **JavaScript** - Interactivité et logique côté client
- **Font** - Georgia, Trebuchet MS, Arial

## 📂 Structure du Projet

```
portfolio/
├── index.html          # Page principale avec le contenu
├── base.css            # Styles globaux, header, footer, responsive
├── sections.css        # Styles des sections (hero, about, skills, projects)
├── parcours.css        # Styles timeline, expériences, contact
├── script.js           # JavaScript pour interactivité
└── README.md           # Documentation du projet
```

## 🚀 Installation & Usage

### Prérequis
- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Aucune dépendance externe requise

### Installation
1. Clonez ou téléchargez le projet
2. Ouvrez `index.html` dans votre navigateur
3. Explorez le portfolio !

### Fonctionnalités JavaScript
- **Mode Thème** : Bouton 🌙 en haut à droite pour basculer entre mode sombre et clair
- **Menu Mobile** : Menu burger adapté pour les appareils mobiles
- **Scroll Reveal** : Éléments animés lors du scroll
- **Navigation Active** : Mise en surbrillance du lien de navigation selon la section visible
- **Barres de Compétences** : Animations de remplissage au scroll

Le portfolio est entièrement responsive :
- **Desktop**: Layout multi-colonnes optimisé

Breakpoint principal : `768px`

## 🔧 Personnalisation

### Modifier les informations
Editez les sections dans `index.html` :
- Nom et titre
- Biographie
- Compétences et niveau
- Projets et descriptions
- Diplômes et expériences
- Coordonnées de contact

### Modifier les styles
- `base.css` : Styles globaux et variables CSS
- `sections.css` : Sections spécifiques
- `parcours.css` : Timeline et contact

### Modifier les animations
Les animations sont définies dans `script.js` :
- `IntersectionObserver` pour le scroll reveal
- Transitions CSS dans les fichiers style
